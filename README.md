## Installation
WEB INSTALL (**fastest**):
```powershell
irm https://raw.githubusercontent.com/DubskySteam/WinFlux/main/main.ps1 | iex
```

LOCAL INSTALL:
```
git clone https://github.com/DubskySteam/WinFlux.git
cd WinFlux
.\main.ps1
```
